# Introdução ao pensamento computacional

# Pensamento computacional

## Decomposição

- Pegar um problema com uma certa complexibilidade e dividir-lo de uma forma que consiga encontrar problemas menores e resolviveis.

## Reconhecimento de padrões

- Identificar padrões ou tendências / similaridades e diferenças entre os problemas

## Abstração

- Estrapolar o conceito do problema para uma forma generalista (pego do mundo concreto e levo para o das ideias)

Design de algorítmos → Automatizar, definir passo a passo a solução do problema

## Habilidades complementares

### Raciocínio lógico

- É uma forma de pensamento estruturado, que permite encontrar a solução e a conclusão ou determinar a resolução de um problema
    
    ### Classificação:
    
    - Indução - Leis e teorias
    - Dedução - Previsões e explicações
    - Abdução - Conclusão → Premissa (não necessáriamente vai ser verdadeira)

### Aperfeiçoamento

- A partir de uma solução, determinar pontos de melhora e refinamento
    - Encontrarsolução eficiente
    - Otimizar processos
    - Simplificar linhas de código
    - Funções bem definidas

# Lógica de programação

## Lógica

- Parte da filosofia que trata das formas de pensamento em geral (dedução, indução, hipótese, …) e das operações intelectuais que visam à determinação do que é verdadeiro ou não

# Fundamentos de algoritmos

## Vetores e matrizes

### Vetores

- Sequencia de tipos de dados com tamanho pre-fixado que irá receber N valores

### Matriz

- Tabela organizadas em linhas e colunas no formato m x n

# Linguagem de programação

- O que é? Método padronizado composto por um conjunto de regras sintáticas e semânticas de implementação de um código fonte

## Caracteristicas de um programa

- Legibilidade
- Redigibilidade
- Confiabilidade
- Custo

### Análise de código

- Lexica = Particionar → Classificar → Eliminar
- Sintática = Corretudo do programa
- Semântica = Lógica do programa

### Paradigmas de programação

- Estruturado = Ênfase na sequência
- Orientação ao  objeto =  Análogo ao mundo real
